# Mobile AI Utility App

This repository contains a Flutter-based mobile application that provides various AI-powered utilities across audio, text, image, data, automation, and logic domains. It is designed to be a comprehensive toolkit for mobile users to perform AI tasks conveniently.

## Features
- Audio processing tools (trimming, noise estimation, silence detection, etc.)
- Text analysis (grammar checking, sentiment analysis, summarization, etc.)
- Image manipulations (filters, cropping, resizing, etc.)
- Data analysis utilities (outlier detection, normalization, summaries, etc.)
- Automation and logic utilities (rule engines, decision trees, etc.)

## Getting Started

Clone the repository and run `flutter pub get` to install dependencies. Use `flutter run` to start the app on your desired platform.
